# CrudFilmes

Sistema de gerenciamento de filmes desenvolvido em PHP puro com padrão MVC e PDO pra exercitar crud

## Funcionalidades
- Cadastro, edição e exclusão de filmes
- Upload de capa para cada filme
- Visualização em catálogo de cards

## Feito em
- PHP
- MySQL
- Bootstrap 5
- Docker
- Deployado na render

## OBS
- Por enquanto as imagens uploadadas se perdem ao recarregar a página, apenas as que estão no repositório são mantidas, pois a Render não possui permanência de arquivos

teste: https://crudfilmes-xxeo.onrender.com/
